1. บุหรี่ ยาสูบ Vape และอุปกรณ์สูบไอ (หน่วยงาน: กรมสรรพสามิต + ด่านศุลกากร)
บุหรี่ไฟฟ้า (E-Cigarette / Vape) — ห้ามนำเข้าโดยเด็ดขาด ตามประกาศกระทรวงพาณิชย์ พ.ศ. 2557 (ของต้องห้าม) มีโทษทั้งจำและปรับ
บุหรี่ธรรมดา ยาเส้น ซิการ์ — ต้องขอใบอนุญาตนำเข้าจากกรมสรรพสามิต + เสียภาษีสรรพสามิตอัตราสูง (สูงสุดเกิน 80%)
น้ำยาบุหรี่ไฟฟ้า / pod / coil — ติดของต้องห้ามเช่นเดียวกับ Vape
ผู้นำเข้าเชิงพาณิชย์ต้องมีใบทะเบียนผู้ค้ายาสูบและใบ อย. ก่อน

2. โดรน / Drone / UAV (หน่วยงาน: กสทช. + CAAT + กรมการปกครอง)
ต้องลงทะเบียนกับ CAAT (สำนักงานการบินพลเรือนแห่งประเทศไทย) ก่อนนำเข้า
ต้องขอใบอนุญาตจาก กสทช. (เพราะมีคลื่นวิทยุ Wi-Fi/2.4GHz/5GHz)
โดรนหนัก > 2 กก. หรือใช้เชิงพาณิชย์ ต้องขออนุญาต กรมการปกครอง เพิ่มเติม
นำเข้าไม่ขออนุญาตล่วงหน้า = ติดด่านเก็บไว้ที่คลังศุลกากร รอ Re-Export หรือยึด

3. เครื่องตรวจจับสัญญาณ / Wi-Fi Repeater / Bluetooth / RF Device (หน่วยงาน: กสทช.)
ทุกอุปกรณ์ที่มีคลื่นวิทยุ Wi-Fi / Bluetooth / Zigbee / RFID — ต้องผ่าน กสทช.
เครื่องตรวจจับสัญญาณ Wi-Fi / Bug Detector / สัญญาณรบกวน (Jammer) — Jammer ห้ามนำเข้าโดยเด็ดขาด
สมาร์ทวอทช์ / สมาร์ทแบนด์ / Smart Home / กล้องวงจรปิด IP Camera — ต้องเช็คเครื่องหมาย กสทช. หรือยื่นขอ Type Approval ก่อนนำเข้า
นำเข้าจำนวนน้อย (เพื่อใช้เอง) = ใช้ใบประกาศตนเองได้ แต่นำเข้าเชิงพาณิชย์ต้องเซอร์ติฟายเครื่อง

4. อาหารเสริม / ยา / เครื่องสำอาง / เครื่องมือแพทย์ (หน่วยงาน: อย. - FDA)
ยา — ต้องใบอนุญาตนำเข้ายา + ทะเบียนตำรับยา (ขอผ่าน อย.)
อาหารเสริม / Vitamin / Collagen — ต้องเลข อย. ก่อนนำเข้าเชิงพาณิชย์
เครื่องสำอาง (Skincare / Makeup) — ต้องจดแจ้งเลขใบรับจดแจ้งเครื่องสำอาง
ผลิตภัณฑ์ที่อ้างสรรพคุณรักษาโรค — เข้าข่ายยา ห้ามนำเข้าโดยไม่มีใบอนุญาตยา
เครื่องมือแพทย์ (Medical Device) — ต้องใบอนุญาตเฉพาะแยกอีกประเภท

5. เครื่องใช้ไฟฟ้า ของเล่นเด็ก หมวกกันน็อค วัสดุก่อสร้าง (หน่วยงาน: มอก. - TISI)
สินค้าควบคุม มอก. — ต้องมีเครื่องหมาย มอก. ติดสินค้า + ใบรับรองจากผู้ผลิต/นำเข้า
เครื่องใช้ไฟฟ้า เช่น พัดลม กาต้มน้ำ เตารีด เครื่องดูดฝุ่น — ติด มอก. หมด
ของเล่นเด็ก (Toys) — มอก. เด็ก ต้องผ่านการทดสอบความปลอดภัยและสารเคมีตกค้าง
หมวกกันน็อค / Helmet — มอก. บังคับสำหรับขายในไทย
สายไฟ / ปลั๊ก / เบรกเกอร์ — มอก. ทั้งหมด

6. อาวุธปืน กระสุน ดาบ มีด (หน่วยงาน: กรมการปกครอง + กรมศุลกากร)
อาวุธปืน / กระสุนปืน — ต้องใบ ป.4 (ใบอนุญาตให้นำเข้า) จาก กรมการปกครอง
ดาบซามูไร / มีดดาบยาว — ต้องเช็คประเภทอาวุธก่อน
สเปรย์พริกไทย / สเปรย์ป้องกันตัว — บางชนิดเข้าข่ายอาวุธ ห้ามนำเข้า
อาวุธจำลอง / Airsoft Gun — ต้องขออนุญาตเฉพาะ บางชนิดติดของต้องห้าม

7. สัตว์มีชีวิต อาหารสัตว์ ผลิตภัณฑ์จากสัตว์ (หน่วยงาน: กรมปศุสัตว์ + กรมประมง)
สุนัข แมว ม้า โค กระบือ — ต้องใบอนุญาตนำเข้าสัตว์มีชีวิต + Health Certificate จากประเทศต้นทาง + กักโรค
อาหารสุนัข / แมว / ปลา — ต้องใบ ร.6 จากกรมปศุสัตว์
ปลาสวยงาม / สัตว์น้ำ — กรมประมง ต้องใบนำเข้า + Health Certificate
ผลิตภัณฑ์จากนม / เนื้อสัตว์ / ไข่ — ต้องใบรับรองสุขอนามัยสัตว์
ขนสัตว์ / กระดูก / หนัง — ต้องเช็คก่อนว่าเป็น CITES species ไหม

8. เมล็ดพันธุ์ ผลไม้สด ปุ๋ย ยาฆ่าแมลง (หน่วยงาน: กรมวิชาการเกษตร)
เมล็ดพันธุ์พืช — ต้องใบอนุญาตนำเข้าเมล็ดพันธุ์ + Phytosanitary Certificate
ผลไม้สด / ผลไม้แห้ง / ผัก — ต้อง Phyto + ใบกักกันโรคพืช + บางชนิดต้องผ่านรังสีหรืออบไอน้ำ
ปุ๋ย สารปรับปรุงดิน — ต้องใบอนุญาตจำหน่ายและนำเข้าปุ๋ย
ยาฆ่าแมลง / สารเคมีเกษตร — ต้องใบอนุญาตวัตถุอันตราย (วอ.6)

9. แอลกอฮอล์ ไวน์ เบียร์ สุรา (หน่วยงาน: กรมสรรพสามิต + กรมศุลกากร)
ต้องใบอนุญาตค้าสุรา ส.๒ จากกรมสรรพสามิต
ต้องเสียภาษีสรรพสามิตตามดีกรีและประเภทเครื่องดื่ม (สูงมาก)
ไวน์โซดา / เครื่องดื่มผสมแอลกอฮอล์ — เข้าข่ายควบคุมเช่นกัน
สุราเถื่อน / สุราโฮมเมด — ห้ามนำเข้า

10. โบราณวัตถุ พระพุทธรูป งานศิลปะ (หน่วยงาน: กรมศิลปากร)
พระเครื่อง พระพุทธรูป รูปเคารพ — ต้องใบอนุญาตจากกรมศิลปากรก่อนนำเข้า/ส่งออก
โบราณวัตถุ (อายุเกิน 100 ปี) — ห้ามส่งออกถ้าไม่ได้ขึ้นทะเบียน
งานศิลปะร่วมสมัย / ภาพวาด — ส่วนใหญ่ไม่ต้องขออนุญาต แต่ต้องเช็คก่อนถ้ามูลค่าสูง

⚠️ สินค้า CITES — ของต้องห้าม / ของควบคุมระหว่างประเทศ
งาช้าง / นอแรด / หนังเสือ / กระดองเต่า — ห้ามนำเข้า/ส่งออกโดยเด็ดขาด (มีโทษทางอาญา)
ไม้พะยูง / ไม้ชิงชัน / ไม้สัก (บางสายพันธุ์) — ต้องใบ CITES + ใบเคลื่อนย้ายไม้
นกแก้วหายาก / เต่าหายาก / กล้วยไม้ป่า — CITES Appendix I-III ต้องเช็คก่อน

🛠️ ทำไมการเช็คก่อนสั่งสำคัญที่สุด?
ถ้านำเข้ามาแล้วไม่มีใบอนุญาต — ของจะถูกอายัดที่ด่าน ค่าโกดังบานปลาย วันละหลักพัน-หลักหมื่น
บางสินค้า "ขออนุญาตย้อนหลังไม่ได้" — เช่น Vape, สุราเถื่อน, อาวุธ — ต้องส่งคืนต้นทางหรือถูกยึด
ของกลายเป็น "ของกลาง" = อายัดตามกฎหมาย ไม่สามารถเรียกคืนได้
ใบอนุญาตบางประเภทใช้เวลายื่น 7-30 วัน ต้องเริ่มก่อนของจะมาถึงไทย
