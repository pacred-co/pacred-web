วิธีลงทะเบียนกรมศุล + จับคู่ YY แบบละเอียด
จาก 0 → พร้อมนำเข้าได้จริง ภายใน 1 ชั่วโมง
การนำเข้าสินค้าให้ “ถูกต้อง + ไม่สะดุด”
 สิ่งที่ทุกคนต้องมีก่อนคือ:
👉 การลงทะเบียนกับ กรมศุลกากร
 👉 และการ “จับคู่ YY (Import–Export Matching)” กับชิปปิ้ง
สองขั้นตอนนี้คือ “หัวใจของการเปิดใบขน”
 ถ้าพลาด = งานค้าง / โดนตรวจ / เสียเวลา
แต่ถ้าทำถูกตั้งแต่ต้น
 👉 คุณสามารถเริ่มนำเข้าได้ทันที

📌 PART 1: การลงทะเบียนกรมศุล (e-Customs) คืออะไร?
คือการ “ยืนยันตัวตน” กับระบบศุลกากรว่า:
คุณคือ “ผู้นำเข้า”
คุณมีสิทธิ์ยื่นใบขน
คุณสามารถสำแดงสินค้าได้ถูกต้องตามกฎหมาย
👉 พูดง่ายๆ:
 ถ้าไม่ลงทะเบียน = เปิดใบขนไม่ได้

⚙️ PART 2: ขั้นตอนลงทะเบียนแบบละเอียด
STEP 1: สมัคร User ในระบบ
กรอกข้อมูลพื้นฐาน
อีเมล / เบอร์โทร
ตั้งรหัสผ่าน
👉 ใช้สำหรับ Login เข้าระบบ e-Customs

STEP 2: ยืนยันตัวตน (Identity Verification)
บุคคลธรรมดา:
บัตรประชาชน
นิติบุคคล:
หนังสือรับรองบริษัท
บัตรประชาชนกรรมการ
👉 จุดสำคัญ:
 ข้อมูลต้อง “ตรง 100%”
 ไม่งั้นระบบไม่อนุมัติ

STEP 3: ลงทะเบียนเป็น “ผู้นำเข้า” (Importer Registration)
เลือกสถานะ: Importer
ผูกกับ User ที่สมัคร
👉 เมื่อผ่านขั้นตอนนี้ =
 คุณมีสิทธิ์เปิดใบขนทันที

STEP 4: ขอ Digital Certificate
ใช้เป็น “ลายเซ็นดิจิทัล”
สำหรับยื่นใบขนในระบบ
👉 ไม่มีตัวนี้ = ยิงใบขนไม่ได้

STEP 5: Mapping ระบบ (สำคัญมาก)
เชื่อม 3 อย่างเข้าด้วยกัน:
User
Importer
Certificate
👉 ถ้า Mapping ไม่ครบ
 = ระบบใช้งานไม่ได้จริง

🔗 PART 3: การจับคู่ YY คืออะไร?
YY Matching คือ:
👉 การ “จับคู่ข้อมูล”
 ระหว่าง:
ข้อมูลฝั่งผู้ส่งออก (Supplier)
ข้อมูลฝั่งผู้นำเข้า (คุณ / ชิปปิ้ง)
เพื่อให้ระบบศุลกากร “ตรวจสอบและอนุมัติ”

⚙️ PART 4: ขั้นตอนจับคู่ YY แบบมืออาชีพ
STEP 1: รับเอกสารจาก Supplier
ต้องมี:
Commercial Invoice
Packing List
HS Code
Incoterms

STEP 2: ตรวจข้อมูล (Pre-Check)
เช็คให้ครบ:
เลข Invoice
รายการสินค้า
จำนวน
มูลค่า
Currency
👉 ต้องตรงกับของจริง 100%

STEP 3: ส่งข้อมูลให้ชิปปิ้ง
ชิปปิ้งจะทำ:
ใส่ข้อมูลในระบบ e-Customs
Mapping ข้อมูลเข้ากับ Importer

STEP 4: ทำ YY Matching
ระบบจะตรวจ:
รายการ
ต้องตรง
Invoice No.
100%
HS Code
100%
Quantity
100%
Value
100%

👉 ถ้าผิด = ระบบไม่ Match

STEP 5: ยิงใบขน (Customs Entry)
เมื่อ Match แล้ว:
ยื่นใบขน
ชำระภาษี
เคลียร์สินค้า

⏱ Flow จริง (ตั้งแต่เริ่มจนจบ)
ลงทะเบียน → ยืนยันตัวตน
 → ได้สิทธิ์ Importer
 → ขอ Certificate
 → รับ Invoice
 → จับคู่ YY
 → ยิงใบขน → รับของ

💣 ปัญหาที่เจอบ่อย (ของจริงในหน้างาน)
ลงทะเบียนไม่ผ่าน
Mapping ผิด
YY ไม่ Match
HS Code ผิด
Invoice ไม่ตรง
👉 ผลลัพธ์:
โดนตรวจ
เสียเวลา
ค่าใช้จ่ายเพิ่ม

🚀 ทางลัดสำหรับคนทำธุรกิจ
ความจริงคือ…
👉 ระบบ “ไม่ได้ยาก”
 แต่ “พลาดไม่ได้เลย”
และสิ่งที่กินเวลาที่สุดคือ:
การตั้งค่า
การ Mapping
การจับคู่ YY

🔥 ทำให้จบใน 1 ชั่วโมง (แบบมืออาชีพ)
ให้ PACRED SHIPPING จัดการให้ครบ
เราไม่ได้แค่ “สอน”
 แต่ “ทำให้เสร็จ”

✅ สิ่งที่คุณจะได้
ลงทะเบียนกรมศุลครบ
Setup e-Customs พร้อมใช้
Certificate ใช้งานได้จริง
จับคู่ YY ถูกต้อง
พร้อม “ยิงใบขนทันที”

⚡ จุดต่างของเรา
👉 จาก “เริ่มไม่เป็น”
 → “นำเข้าได้เลย”
ภายใน 1 ชั่วโมง

🎯 เหมาะกับใคร
มือใหม่เริ่มนำเข้า
เจ้าของธุรกิจที่อยากทำเอง
คนที่เคยติดปัญหาเอกสาร
คนที่ต้องการความเร็ว + ความถูกต้อง

📞 ปิดการขายแบบเนียน
ถ้าคุณกำลัง:
งงกับระบบกรมศุล
กลัวทำผิดแล้วเสียเงิน
อยากเริ่มนำเข้า “วันนี้เลย”
👉 ให้ PACRED SHIPPING ดูแลให้ทั้งหมด
คุณแค่ส่งเอกสาร — ที่เหลือเราจัดการให้จบ

💥 สรุปสั้น
ลงทะเบียน = ได้สิทธิ์
YY Matching = ทำให้งานผ่าน
PACRED = ทำให้ทุกอย่างง่าย

💬 ทักมาได้ทันที
 เราพร้อมเปลี่ยนเรื่องยาก
 👉 ให้คุณ “เริ่มนำเข้าได้ภายในวันเดียว”
