# PCS Cargo — Business Flow

**Version:** 1.0 | **Updated:** May 19, 2026

---

## Overview

PCS Cargo ให้บริการนำเข้าสินค้าจากจีนมาไทยผ่าน 3 บริการหลัก

| บริการ | คำอธิบาย | รายได้ |
|--------|-----------|--------|
| **Shopping Service** (ฝากสั่งสินค้า) | ลูกค้าส่ง URL สินค้า PCS สั่งซื้อแทน | Service fee 5% (VIP 3%) |
| **Forwarding Service** (ฝากนำเข้า) | ลูกค้าสั่งเองแล้วให้ PCS นำเข้า | ค่าขนส่งตามน้ำหนัก/ปริมาตร |
| **Payment Service** (ฝากชำระ/โอน) | PCS โอนเงินให้ Supplier ในจีนแทน | Service fee 3% |

## 1. Shopping Service — Order Flow

```
ลูกค้า                    PCS System                 คลังจีน/ไทย
  │                           │                           │
  │── วางสินค้าในตะกร้า ──────>│                           │
  │   (URL จาก 1688/Taobao)   │ คำนวณราคา + ค่าบริการ     │
  │<── แสดง Price Breakdown ───│                           │
  │                           │                           │
  │── เลือกวิธีชำระ ──────────>│                           │
  │   (Wallet/โอน/Credit)     │                           │
  │                           │                           │
  │  [ถ้าโอนธนาคาร]           │                           │
  │── Upload Slip ────────────>│                           │
  │                           │── Admin ตรวจสลิป           │
  │<── ยืนยันชำระ ─────────────│                           │
  │                           │                           │
  │                           │── สั่งซื้อจากร้านจีน ──────>│
  │                           │   (ภายใน 24 ชั่วโมง)       │
  │                           │<── เลขออเดอร์จีน ──────────│
  │                           │                           │
  │<── SMS: กำลังดำเนินการ ────│                           │
  │                           │                           │
  │                           │<── สินค้าถึงคลังจีน ───────│
  │<── SMS: สินค้าถึงคลังจีน ──│   (ถ่ายรูป + ตรวจ)        │
  │                           │                           │
  │                           │── รวม + ส่งมาไทย ──────────>│
  │<── SMS: กำลังขนส่ง ────────│                           │
  │                           │                           │
  │                           │<── ถึงคลังไทย ─────────────│
  │                           │   (เคลียร์ศุลกากร)         │
  │<── SMS: ออกจัดส่ง ─────────│── ส่ง DHL/Flash/Kerry     │
  │                           │                           │
  │<── [ลูกค้ารับสินค้า] ──────────────────────────────────│
```

### Status ของ Shopping Order

| Code | สถานะ | ความหมาย |
|------|-------|-----------|
| `1` | Draft | อยู่ในตะกร้า |
| `2` | รอชำระเงิน | สร้างออเดอร์แล้ว รอลูกค้าจ่าย |
| `3` | กำลังดำเนินการ | ชำระแล้ว รอสั่งจากร้านจีน |
| `4` | สั่งจากร้านจีนแล้ว | กำลังรอสินค้า |
| `5` | ถึงคลังจีน | สินค้าอยู่ที่โกดังจีน |
| `6` | ส่งมาไทย | อยู่ระหว่างขนส่ง |
| `7` | ถึงไทย | อยู่ที่โกดังไทย |
| `8` | กำลังจัดส่ง | อยู่กับขนส่งไทย |
| `9` | สำเร็จ | ลูกค้าได้รับแล้ว |
| `0` | ยกเลิก | ยกเลิกแล้ว |

## 2. Forwarding Service — Import Flow

```
ลูกค้า                    PCS System                 คลังจีน
  │                           │                           │
  │── สร้างรายการนำเข้า ───────>│                           │
  │   (เลข Tracking + ขนาด)   │ คำนวณค่าใช้จ่ายเบื้องต้น  │
  │<── แสดงราคาประเมิน ────────│                           │
  │                           │                           │
  │                           │<── สินค้าถึงคลังจีน ───────│
  │                           │   ชั่งน้ำหนัก / วัดจริง    │
  │                           │   ถ่ายรูป + ตรวจสอบ        │
  │                           │                           │
  │  [ถ้าราคาเพิ่มเกิน 10%]   │                           │
  │<── แจ้งลูกค้ายืนยัน ───────│                           │
  │── ยืนยัน/ยกเลิก ──────────>│                           │
  │                           │                           │
  │                           │── แพ็ก + จัดส่งมาไทย ──────>│
  │                           │   (เรือ/เครื่องบิน/Express)│
  │                           │                           │
  │                           │── เคลียร์ศุลกากร          │
  │                           │── ถึงคลังไทย              │
  │                           │                           │
  │<── Invoice + รอชำระ ───────│ (ราคาจริงหลังชั่งจริง)    │
  │── ชำระเงิน ───────────────>│                           │
  │<── ยืนยัน + ออกจัดส่ง ────│                           │
  │                           │                           │
  │<── [ลูกค้ารับสินค้า] ──────────────────────────────────
```

### Status ของ Forwarding Order

| Code | สถานะ | ความหมาย |
|------|-------|-----------|
| `1` | Draft | กรอกข้อมูลยังไม่ครบ |
| `2` | รอสินค้า | กรอก Tracking แล้ว รอสินค้าถึงคลังจีน |
| `3` | ถึงคลังจีน | รับสินค้าแล้ว กำลังแพ็ก |
| `4` | ถึงไทย | ผ่านศุลกากร อยู่คลังไทย |
| `5` | รอชำระ | ออก Invoice รอลูกค้าจ่าย |
| `6` | พร้อมจัดส่ง | ชำระแล้ว กำลังเตรียมส่ง |
| `7` | กำลังจัดส่ง | อยู่กับขนส่งไทย |
| `8` | สำเร็จ | ลูกค้าได้รับแล้ว |

## 3. Payment Service — Transfer Flow

```
ลูกค้า                    PCS System                 จีน
  │                           │                           │
  │── สร้างคำขอโอนเงิน ────────>│                           │
  │   (จำนวน CNY + ผู้รับ)    │ คำนวณ THB + ค่าบริการ 3%  │
  │<── ยืนยันราคา ─────────────│                           │
  │                           │                           │
  │                           │── Admin ตรวจสอบ 24 ชม.    │
  │                           │   (เอกสาร + ความเสี่ยง)   │
  │                           │                           │
  │<── แจ้งอนุมัติ/ปฏิเสธ ─────│                           │
  │                           │                           │
  │── ชำระเงิน ───────────────>│                           │
  │                           │── โอน Alipay/WeChat/Bank ─>│
  │                           │<── หลักฐานการโอน ──────────│
  │<── แจ้งสำเร็จ + แนบหลักฐาน─│                           │
```

## 4. Wallet Flow

```
เติมเงิน (Top-up):
ลูกค้า โอนธนาคาร/QR → Upload Slip → Admin ตรวจ → เติม Wallet อัตโนมัติ

ชำระออเดอร์:
Checkout → ใช้ยอด Wallet → หัก Wallet → อัปเดตยอดคงเหลือ

ถอนเงิน (Withdraw):
ลูกค้าขอถอน → ทีมการเงินตรวจ → โอนกลับ → อัปเดต Wallet

เครดิต VIP:
สั่งสินค้า → หัก Credit Limit → ครบกำหนดจ่าย → ชำระ → คืน Limit
```

### Credit VIP Rules

| เงื่อนไข | ค่า |
|----------|-----|
| อายุบัญชีขั้นต่ำ | 30 วัน |
| ออเดอร์ขั้นต่ำ | 10 ออเดอร์ |
| ยอดสะสมขั้นต่ำ | 50,000 THB |
| วงเงินเริ่มต้น | 2× ค่าเฉลี่ยต่อออเดอร์ (สูงสุด 10,000 THB) |
| วงเงินสูงสุด | 100,000 THB |
| ดอกเบี้ยค้างชำระ ≤7 วัน | 2% |
| ดอกเบี้ยค้างชำระ 8–14 วัน | 5% |
| ดอกเบี้ยค้างชำระ >14 วัน | 10% |

## 6. Cancel & Refund Policy

| สถานะออเดอร์ | Shopping Service | Forwarding Service |
|-------------|-----------------|-------------------|
| ยังไม่ชำระ | ยกเลิกได้ทันที | ยกเลิกได้ทันที |
| กำลังดำเนินการ | อาจมีค่าธรรมเนียม | ยกเลิกได้ถ้าสินค้ายังไม่ถึงคลังจีน |
| ร้านจีนส่งแล้ว / ถึงคลังจีนแล้ว | **ยกเลิกไม่ได้** | อาจมีค่าส่งกลับ |
| กำลังขนส่ง | ไม่สามารถยกเลิก | ไม่สามารถยกเลิก |
