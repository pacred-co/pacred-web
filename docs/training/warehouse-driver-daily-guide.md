# 📦🚚 คู่มือสอนงานประจำวัน — พนักงานโกดัง + คนขับรถ (Pacred)

> สำหรับ **ภูม ใช้สอนพี่พนักงานโกดัง + คนขับรถ** ว่าวันนึงต้องทำอะไร · วางแผนอะไร · กดหน้าไหนในระบบ.
> เขียนแบบภาษาพูด อ่านง่าย · อ้างอิงหน้าจริงในระบบ Pacred (admin).
> 🟡 ส่วนที่เป็น **กติกาบริษัท** (เวลาเข้างาน-เลิกงาน · กี่รอบ/วัน · ใครทำหน้าที่ไหน · ผังโกดัง) = ภูม/พี่ป๊อป เติมเอง — ผมเขียนเฉพาะ **flow ของระบบ** ที่รู้แน่ๆ.

---

## 🧭 ภาพรวมก่อน — ของเดินทางยังไง (สถานะ 1→7)

ของลูกค้าวิ่งจากจีน → โกดังไทย → ถึงมือลูกค้า ผ่าน **7 สถานะ** (เรียกว่า "สถานะฝากนำเข้า"):

| สถานะ | ชื่อ | ใครดูแลช่วงนี้ |
|---|---|---|
| **1** | รอเข้าโกดังจีน | (จีน/ระบบ sync) |
| **2** | ถึงโกดังจีนแล้ว | (จีน/ระบบ sync) |
| **3** | กำลังส่งมาไทย | (ขนส่ง) |
| **4** | **ถึงไทยแล้ว** | 🏭 **โกดัง** เริ่มที่นี่ |
| **5** | **รอชำระเงิน** | 🏭 โกดัง + บัญชี |
| **6** | **เตรียมส่ง** (ลูกค้าจ่ายแล้ว) | 🏭 โกดัง → 🚚 คนขับ |
| **7** | **ส่งแล้ว** | 🚚 **คนขับ** จบที่นี่ |

> 💡 **โกดังทำงานช่วง 4→5→6 · คนขับทำงานช่วง 6→7.** ทุกหน้าในระบบจะมีป้ายบอก "ขั้นต่อไปทำอะไร" อยู่ใต้สถานะ — ดูป้ายนั้นได้เลย ไม่ต้องจำ.

---

## 🏭 ส่วนที่ 1 — พนักงานโกดัง: วันนึงทำอะไร

### ☀️ ตอนเช้า — เช็คของที่จะเข้าวันนี้
1. เปิดหน้า **รายการฝากนำเข้า** → `แอดมิน > บริการนำเข้า > รายการนำเข้า` (`/admin/forwarders`)
2. ดูว่ามีของสถานะ **"กำลังส่งมาไทย (3)"** / **"ถึงไทยแล้ว (4)"** กี่ชิ้น → วางแผนว่าวันนี้ต้องชั่ง/วัด/แพ็คประมาณเท่าไร
3. ของจาก MOMO/แต้ม ระบบ **ดึงเข้าให้อัตโนมัติ** (ไม่ต้องคีย์เอง) — แค่มาเช็คว่าครบ

### 📥 ขั้นที่ 1 — ของถึงโกดังไทย → **สแกนรับเข้า**
- เปิดหน้า **สแกนบาร์โค้ด** → `แอดมิน > สแกนรับสินค้า` (`/admin/barcode`)
- สแกน/พิมพ์ **เลขแทรกกิ้ง** ของแต่ละกล่อง → ระบบเด้งสถานะเป็น **"ถึงไทยแล้ว (4)"** ให้เอง
- 👉 *แปลว่า: ของชิ้นนี้อยู่ในมือเราแล้ว*

### ⚖️ ขั้นที่ 2 — **ชั่งน้ำหนัก + วัดขนาด** → คีย์เข้าระบบ
- เปิดรายการนั้น → กด **แก้ไข** (`/admin/forwarders/[เลขที่]/edit`)
- ใส่ **น้ำหนัก (กก.)** + **กว้าง × ยาว × สูง (ซม.)**
- ระบบ **คิดค่าขนส่งให้อัตโนมัติ** ตามเรทของลูกค้าคนนั้น
- กดบันทึก → ถ้าเซลตั้งเรทไว้แล้ว ระบบเด้งเป็น **"รอชำระเงิน (5)"** ให้เอง
- ⚠️ ถ้ายังไม่มีเรท (เช่นลูกค้าใหม่) ระบบจะค้างไว้ + ขึ้นเตือน "ตั้งเรทก่อน" — แจ้งเซล

### 📦 ขั้นที่ 3 — **เช็คตู้ / ปิดตู้** (เฉพาะของมาเป็นตู้คอนเทนเนอร์)
- เปิดหน้า **รายงานตู้** → `/admin/report-cnt`
- เช็คว่าของในตู้ **ครบตาม packing list** ของแต้ม/MOMO ไหม (ระบบเทียบให้)
- ของขาด/เกิน → flag ไว้ แจ้งเดฟ/บัญชี

### 🧾 ขั้นที่ 4 — **วางบิล** (ส่งให้ลูกค้าจ่าย)
- เปิดหน้า **ตรวจ+วางบิล** → `/admin/forwarder-check`
- ติ๊กรายการที่พร้อม → กดวางบิล → ระบบส่งบิลให้ลูกค้า (รอชำระ)

### 💳 ขั้นที่ 5 — **ลูกค้าจ่ายแล้ว → เตรียมส่ง (6)**
- พอลูกค้าจ่าย ระบบเด้งเป็น **"เตรียมส่ง (6)"** ให้เอง
- ของที่สถานะ 6 = **พร้อมแพ็ค + ส่ง**

### 🏷️ ขั้นที่ 6 — **พิมพ์ป้ายติดกล่อง**
- ในรายการ กด **พิมพ์ป้าย** → หน้า `/admin/forwarders/print`
- มี 2 แบบ: **ป้ายหน้ากล่อง** (เลขลูกค้า+QR+บาร์โค้ด) · **ป้ายที่อยู่** (ผู้รับ+ที่อยู่+ขนส่ง)
- ตั้งเครื่องพิมพ์: **กระดาษ 100×75 · Margins None · Scale 100% · Darkness สูง** แล้วพิมพ์ → ติดกล่อง

### 🚚 ขั้นที่ 7 — **มอบงานคนขับ / จัดรับเองหน้าโกดัง**
- เปิดหน้า **มอบงานคนขับ** → `/admin/drivers/new`
- มี 3 แท็บ:
  - **มอบคนขับ** (Pacred ส่งเอง) → เลือกของ + เลือกคนขับ + กดมอบ → ไปเข้ารอบคนขับ
  - **รับเองหน้าโกดัง** → ลูกค้ามารับเอง → กดส่งสำเร็จตอนลูกค้ามารับ
  - **Express** (ขนส่งภายนอก Flash/Kerry/J&T) → จัดส่งเจ้าอื่น

### 📋 พนักงานโกดัง — **ต้องวางแผนอะไร**
- 🔢 **ดูปริมาณของเข้าวันนี้** (สถานะ 3-4) → จัดคนชั่ง/วัดให้ทัน
- 🧹 **เคลียร์ของค้าง** สถานะ 4 (ยังไม่ชั่ง) + สถานะ 5 (รอลูกค้าจ่าย — ตามบัญชี)
- 📦 **เตรียมพื้นที่/ตู้** ตามของที่จะเข้า
- 🏷️ **เตรียมป้าย+เครื่องพิมพ์** ให้พร้อม (กระดาษ 100×75 · Darkness)
- 🚚 **ประสานคนขับ** — รอบบ่าย/พรุ่งนี้มีของกี่จุด ส่งโซนไหน

## 🚚 ส่วนที่ 2 — คนขับรถ: วันนึงทำอะไร

### ☀️ ตอนเช้า — รับรอบงาน
- เปิดหน้า **งานคนขับ (มือถือ)** → `/admin/drivers/work`
- ดู **รายการที่โกดังมอบหมาย** ให้รอบนี้ → มีกี่จุด ส่งที่ไหนบ้าง
- แต่ละจุดบอก: **ชื่อลูกค้า + ที่อยู่ + เบอร์โทร + จำนวนกล่อง + เลขแทรกกิ้ง**

### 🗺️ ขั้นที่ 1 — **แพลนเส้นทาง**
- ระบบ **เรียงลำดับส่งให้แล้ว** (ใกล้โกดัง→ไกล · ตามเขต/โซน กทม.-ปริมณฑล)
- ดูลำดับ → วางเส้นทางตามนั้น (ปรับได้ถ้ารู้ทางลัด)

### 📦 ขั้นที่ 2 — **รับของจากโกดัง**
- เช็ค **จำนวนกล่อง** + **ป้ายติดกล่อง** (เลขลูกค้า/แทรกกิ้ง) ให้ตรงกับรายการในมือถือ
- ครบ → ขึ้นรถ ออกส่ง

### ✅ ขั้นที่ 3 — **ส่งสำเร็จ**
- ถึงลูกค้า → ส่งของ → **ถ่ายรูป** (หน้าบ้าน/ผู้รับ/ของที่ส่ง)
- กด **"ส่งสำเร็จ"** + แนบรูป → สถานะเด้งเป็น **"ส่งแล้ว (7)"**
- 👉 *ลูกค้าเห็นในระบบทันทีว่าของถึงแล้ว*

### ❌ ขั้นที่ 4 — **ส่งไม่ได้** (ลูกค้าไม่อยู่/ติดต่อไม่ได้/ที่อยู่ผิด)
- กด **"ส่งไม่สำเร็จ"** + **บันทึกเหตุผล** (เช่น "โทรไม่รับ", "บ้านปิด", "ที่อยู่ผิด")
- เหตุผลจะโชว์ให้โกดัง/แอดมินเห็น → จัดรอบส่งใหม่

### 💰 ขั้นที่ 5 — **เก็บเงินปลายทาง (COD)** — ถ้ามี
- ออเดอร์ที่เป็น **"เก็บปลายทาง"** → เก็บเงินจากลูกค้าตอนส่ง → บันทึก/ส่งมอบตามกติกาบริษัท

### 🏁 ขั้นที่ 6 — **ปิดรอบ**
- ส่งครบทุกจุดในรอบ → ระบบ **เด้งรอบเป็น "สำเร็จ" อัตโนมัติ** (ไม่ต้องกดปิดเอง)
- มีจุดส่งไม่ได้ → รอบจะค้างไว้ให้จัดใหม่

### 📋 คนขับ — **ต้องวางแผนอะไร**
- 🕐 **เช็ครอบ** เช้า/บ่าย มีกี่จุด → คำนวณเวลา
- 🗺️ **วางเส้นทางตามโซน** (อย่าวิ่งย้อนไปย้อนมา) — ดูลำดับที่ระบบจัดให้
- 🚗 **เผื่อเวลารถติด** (ช่วงเร่งด่วน) + เผื่อเวลาหาบ้าน
- 📞 **โทรลูกค้าก่อนถึง** (กันลูกค้าไม่อยู่ = เสียเที่ยว)
- 💵 **เตรียมเงินทอน** ถ้ามีจุดเก็บปลายทาง (COD)
- 📸 **อย่าลืมถ่ายรูปทุกจุด** (เป็นหลักฐานส่งสำเร็จ)

## 🔑 กฎทองที่ทุกคนต้องจำ
1. **อัพเดทสถานะทุกครั้งที่ทำเสร็จ** — สแกนเข้า / ชั่ง / ส่งสำเร็จ → กดในระบบ ลูกค้าเห็นเรียลไทม์
2. **ถ่ายรูปเป็นหลักฐาน** — ของถึงไทย / ส่งสำเร็จ / ของเสียหาย
3. **มีปัญหา = บันทึกในระบบ** อย่าเก็บไว้ในหัว (ของไม่ครบ / ส่งไม่ได้ / ที่อยู่ผิด)
4. **ดูป้าย "ขั้นต่อไป"** ใต้สถานะแต่ละรายการ — ระบบบอกอยู่แล้วว่าต้องทำอะไรต่อ

## 🟡 สิ่งที่ ภูม/พี่ป๊อป ต้องเติม (กติกาบริษัท — ผมไม่รู้)
- เวลาเข้า-เลิกงาน · กี่รอบส่ง/วัน · เวลาตัดรอบ
- ใครรับผิดชอบหน้าไหน (ใครสแกน / ใครชั่ง / ใครมอบงานคนขับ)
- ผังโกดัง / โซนวางของ / รหัส location (เช่น A1, B2)
- กติกา COD (เก็บเงินแล้วส่งมอบใคร เมื่อไร)
- เบอร์ติดต่อฉุกเฉิน (ลูกค้าไม่อยู่ → โทรหาใคร)
